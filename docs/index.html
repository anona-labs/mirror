Mirror

A live, styled HTML view of your Claude Code conversation. Keep working in the terminal. Mirror gives you a link that updates every turn.

Why it exists

Read your session, do not scroll it.

01

It costs nothing to run

Mirror never calls a model. It renders the transcript your session already writes to disk, so it rides on the subscription you already pay for. No keys, no metering.

02

It never leaves your machine

The server listens on 127.0.0.1 only. Your transcript, with its secrets, file contents, and tool output, stays on localhost. Nothing is uploaded.

03

It updates as you work

The page refreshes after every turn over Server-Sent Events and keeps your scroll position, so a long session stays readable instead of vanishing up the terminal.

04

It is almost nothing

Pure Python standard library on the server. No pip, no npm, no build step. Clone it, point Claude Code at it, done.

Install

Two lines to a live link.

# clone, then point Claude Code at it
git clone https://github.com/anoopbhat44/mirror.git
claude --plugin-dir /path/to/mirror

Or install it permanently through a local marketplace:

claude plugin marketplace add /path/to/mirror
claude plugin install mirror

Use it

There is no step four.

  1. Start Claude Code the way you always do.

    Mirror's hooks start a tiny local server in the background.

  2. Open the link it prints.

    You will see 🪞 Mirror live view: http://localhost:7842

  3. Watch it render.

    The conversation appears and updates after every turn.
